# Inspect-Me Landing Page

Coming-soon marketing site for the **Inspect-Me** light inspection-class ROV training simulator.

## Quick start

```bash
py -m http.server 8080
```

Open `http://localhost:8080`.

## When the game is ready

Set `comingSoon: false` in `config.js` and add your build URLs. Platform buttons will activate automatically.

## Assets

Game screenshots live in `assets/images/`. Replace them as better captures become available.

## Deploy

Static files — GitHub Pages, Netlify, Vercel, or itch.io (HTML project).
